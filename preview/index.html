AI Traffic Light

Cursor 第一版 · HTML 悬浮 UI 预览

悬浮窗预览
绿灯：Agent 结束或未工作
黄灯：推理 / thinking 阶段
红灯：工具执行中（Shell、Read、Write 等）

第一版方案（仅 Cursor）

UI 全部放在 HTML 里；macOS 壳只负责「透明置顶窗口 + 加载本地 HTML + 提供状态接口」。

1. Cursor Hooks 用户级 `~/.cursor/hooks.json` 监听 Agent 生命周期，写入统一状态文件。
2. 状态文件 `~/Library/Application Support/ai-traffic-light/state.json`
3. 悬浮 App 透明 WebView 加载 `ui/widget.html`，轮询 `/state` 或监听文件变化。

状态机

  • beforeSubmitPrompt / afterAgentThought → thinking
  • preToolUse → running
  • postToolUse 且无后续 thinking → 回到 thinking
  • stop / sessionEnd → idle

Hook 映射

{
  "version": 1,
  "hooks": {
    "beforeSubmitPrompt": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh thinking" }],
    "afterAgentThought": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh thinking" }],
    "preToolUse": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh running" }],
    "postToolUse": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh thinking" }],
    "stop": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh idle" }],
    "sessionEnd": [{ "command": "~/.cursor/hooks/ai-traffic-light.sh idle" }]
  }
}
